# VantaTrack Ad Engine - Project Brief

## Core Components
1. **Ad Engine**: For Bangladesh news portals (Daily Star, Dhaka Tribune, etc.)
2. **Media Buying Tracking**: Google Ads + Facebook Ads + Portal campaigns
3. **Unified Dashboard**: For agencies and clients

## Key Features
- Dual data input: API integration + Manual CSV upload
- Multi-tenant access control (clients see only their platforms)
- Bangladesh market optimization (BDT currency, 3G optimization)
- Premium UI with deep blue (#1e40af) and emerald (#059669) branding

## Technical Stack
- Backend: Node.js + Express + PostgreSQL
- Frontend: React with premium UI components
- Database: vantatrack_* table naming
- APIs: Google Ads API + Facebook Marketing API + Manual uploads

## Client Access Levels
- Portal Only, Google Only, Facebook Only
- Combined packages: Google+Facebook, Portal+Google, Portal+Facebook
- Full Suite: All three platforms unified

## Timeline
- Launch Target: September 26, 2024
- Development: 7-day sprint approach
